Class MapViewImageCache

Cache images wrapped into ImageItems for a MapView. An image may have multiple names in a theme, the MapViewImageCache will take care of that. Registering multiple images with the same name is invalid.

The MapViewImageCache uses a global ImageCache to actually store (and generate) the image data.
